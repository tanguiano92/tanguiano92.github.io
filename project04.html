Title: Coding Fun


Medium: code, css


Dimensions: Unknown


Date Created: May 9, 2019




I created this piece using in-line code and by basically playing around with RBG
codes. I aspired to make something in relation to the basic RBG, CYMK and black and white
colors in relation to the color systems used in web design and so on. I also was experimenting
with the sizing and trying to make each ellipse look as though they are in different stages of
creation (i.e. beginner, intermediate and advanced). Although this code painting is quite simple,
it was a long process to get the image to the point that it is at now. I plan to keep working on
this piece and adding more to it and play around with the concept a bit further.
